# all-around keyboards
a web component for round pianos, based on https://bl.ocks.org/mbostock/5723d93e4f617b542991

### example:
```html
<all-around-keyboard  notes-in-octave="12"
                      raised-keys="[2,4,7,9,11]"
                      sweep=270 octaves="2"
                      thickness="100" width="500"
                      overlapping="2.75">
</all-around-keyboard>
```
